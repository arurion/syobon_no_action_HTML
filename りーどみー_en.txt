・Controls

Left / Right　…　Move

Up or Z key　…　Jump

Hold Space　…　2x Speed

O key  …　Self-destruct

ESC　…　Quit

F1　…　Return to Title

1–4 keys (at Title)　…　Select Stage

0 key　…　Mystery Dungeon (Warning: May contain bugs)
